# esp32-as-uart-ttl-converter
esp32 as uart ttl converter with micropython. 
Use TX2 and RX2 pin(it worked for me).
|    | UART0 | UART1 | UART2 |
|----|-------|-------|-------|
| tx | 1     | 10    | 17    |
| rx | 3     | 9     | 16    |
